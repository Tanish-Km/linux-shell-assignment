# linux-shell-assignment

## Project overview
This repository contains three beginner-friendly shell scripts that demonstrate:
- Creating a timestamped backup of a directory.
- Monitoring CPU and memory usage and logging it to a file.
- Downloading a file from the internet (using curl or wget).

## Files
- `backup_simple.sh` — Create a tar.gz backup of a directory with a timestamp.
- `monitor_simple.sh` — Log load average and memory usage periodically to a log file.
- `download_simple.sh` — Download a file from a URL into a directory.
